# Palindrome Checker App

A comprehensive Java application demonstrating multiple approaches to checking palindromes, progressing from basic implementations to optimized solutions.

---

## 📌 Overview

This project contains multiple use cases (UC01–UC13) implementing different palindrome-checking techniques. Each use case builds upon the previous one with improvements in logic, efficiency, and data structures.

## 🚀 Use Cases

| UC   | Title                    | Approach                      |
| ---- | ------------------------ | ----------------------------- |
| UC01 | Welcome Page             | Basic introduction            |
| UC02 | Hardcoded Palindrome     | Static input validation       |
| UC03 | String Reverse           | Simple string reversal        |
| UC04 | Character Array          | Array-based checking          |
| UC05 | Stack-Based              | Stack data structure          |
| UC06 | Queue & Stack            | Dual structure approach       |
| UC07 | Deque Optimization       | Optimized deque usage         |
| UC08 | Linked List              | Linked list implementation    |
| UC09 | Enhanced Validation      | Input validation improvements |
| UC10 | Case Handling            | Case-insensitive comparison   |
| UC11 | Special Characters       | Ignore spaces/symbols         |
| UC12 | Performance Optimization | Efficient algorithm           |
| UC13 | Final Version            | Fully optimized solution      |

## 🛠️ Getting Started

### ✅ Prerequisites

* Java 8 or higher
* Any Java IDE (IntelliJ / VS Code / Eclipse)

---

### ▶️ Running the Application

#### Using IDE

1. Open the project in your IDE
2. Navigate to `App/src/`
3. Run any `UCXXPalindromeCheckerApp.java` file

---

#### Using Command Line

```
cd App/src
javac UC01PalindromeCheckerApp.java
java UC01PalindromeCheckerApp
```
